# Exemplos de Documentos BSON

Este diretório contém exemplos de documentos BSON (Binary JSON) usados em bancos de dados NoSQL, como o MongoDB.

## Coleções

Coleções são conjuntos de documentos em JSON/BSON. Elas permitem armazenar e recuperar dados de forma eficiente, sem esquema rígido.

- **Semelhante a tabelas**: cada documento pode ter estrutura diferente.  
- **Semelhante a pastas**: documentos são como arquivos dentro delas.
